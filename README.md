# Carmine
Carmine is network of projects focused on Minecraft server

## Modules

This monorepo contains three gradle modules that work together:

### **[Raven](./Raven/RAVEN.md)**
Custom Minecraft server built on [Minestom](https://minestom.net/)

### **[Solace](./Solace/SOLACE.md)**
Discord bot meant to work with Raven

### **[Lumen](./Lumen/LUMEN.md)**
Shared utility library for common stuff

### **[CLI](./CLI/CLI.md)**
Useful cli tool

> See individual module READMEs for setup instructions and documentation.
